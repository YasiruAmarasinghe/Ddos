# Ddos
### Installing

* `pkg update`
* `pkg upgrade`
* `pkg install git`
* `apt install python`
* `apt install python2`
* `apt install python3`
* `git clone https://github.com/mishakorzik/Ultra-DDos`
* `pip3 install requests`


#### Finished you have successfully downloaded Ultra-DDos now to launch!

* `cd Ultra-DDos`
* `python2 main.py`
